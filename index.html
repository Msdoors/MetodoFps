Como Usar o Script Python para Ajustar FPS do Vídeo

Passo 1: Instalar o Termux

Primeiramente, baixe o Termux via F-Droid. O F-Droid é uma loja de aplicativos confiável que oferece versões estáveis do Termux para Android. Importante: não baixe o Termux pela Play Store, pois a versão lá está desatualizada. Você pode obter o Termux pelo link do F-Droid:

https://f-droid.org/packages/com.termux/

Passo 2: Instalar o Python

Após instalar o Termux, abra o terminal do Termux e execute o seguinte comando para instalar o Python 3:

pkg install python

Passo 3: Criar o Script

No Termux, crie o arquivo de script com o seguinte comando:

nano patch_mp4.py

Depois, cole o código Python fornecido no script dentro do arquivo e salve pressionando CTRL + X, depois Y para confirmar e Enter para sair.

Passo 4: Preparar o Vídeo

Envie o vídeo para o diretório do Termux onde o script foi criado. Você pode usar o comando cp ou mover o arquivo manualmente.

Passo 5: Rodar o Script

Agora, você vai rodar o script para ajustar o FPS do seu vídeo. Antes de rodar, é muito importante calcular o fator de escala. Veja como calcular isso:

Como Calcular o Fator de Escala

Verifique o FPS do seu vídeo original (se você não souber, pode usar um reprodutor de mídia que mostra o FPS, como o VLC). Divida 30 pelo FPS do seu vídeo. O resultado será o valor que você precisa passar como argumento para o script.

Exemplo:

  • Se o seu vídeo tem 60 FPS, então: 30 / 60 = 0.5. Execute o comando:
    • python patch_mp4.py input.mp4 output.mp4 0.5
  • Se o seu vídeo tem 90 FPS, então: 30 / 90 = 0.333. Execute o comando:
    • python patch_mp4.py input.mp4 output.mp4 0.333
  • Se o seu vídeo tem 120 FPS, então: 30 / 120 = 0.25. Execute o comando:
    • python patch_mp4.py input.mp4 output.mp4 0.25

Recomendo que o arquivo original tenha 120+ FPS para uma melhor fluidez ao postar no TikTok!

Passo 6: Aplicar o Patch

Quando você rodar o script, ele irá aplicar o patch e ajustar o FPS do seu vídeo. Após o processo ser concluído, o vídeo alterado será salvo como output.mp4.

Postando no TikTok

MUITO IMPORTANTE:

  • Ao postar seu vídeo no TikTok, use o navegador Brave ou Chrome.
  • Carregue o vídeo como privado primeiro e, depois, mude para público.
  • Preferencialmente, faça o upload pelo computador. Se não for possível, use o celular, mas no modo desktop para garantir que o vídeo seja carregado corretamente.

Agora é só curtir seu vídeo fluido e com a edição feita no Termux!
